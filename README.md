# CLI Task Tracker (Java)

A simple **Command-Line Task Tracker** built in **Java** using **OOP concepts** and **CRUD operations**.
This project allows users to manage daily tasks directly from the terminal.

## Features

* Add new tasks (Create)
* View all tasks (Read)
* Mark tasks as completed (Update)
* Delete tasks (Delete)
* Menu-driven CLI interface
* Input validation and exception handling

---

## Tech Stack

* **Language:** Java
* **Concepts Used:**

  * OOP (Classes & Objects)
  * Constructors
  * Encapsulation
  * ArrayList
  * CRUD Operations
  * Exception Handling
  * Loops
  * Switch Case
  * Scanner for User Input

---

## Project Structure

```bash
TaskTracker.java
│
├── Task Class
│   ├── Task Name
│   ├── Completion Status
│   └── Methods for Task Handling
│
└── TaskTracker Class
    ├── Add Task
    ├── View Tasks
    ├── Mark Complete
    ├── Delete Task
    └── Menu System
```

---

## How It Works

### Menu Options

```text
1. Add Task       (CREATE)
2. View Tasks     (READ)
3. Mark Complete  (UPDATE)
4. Delete Task    (DELETE)
5. Exit
```
